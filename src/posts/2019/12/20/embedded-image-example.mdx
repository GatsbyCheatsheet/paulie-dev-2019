import { EmbeddedImage } from "@pauliescanlon/gatsby-theme-gatstats/src/components/EmbeddedImage"
import { Flex, Box } from '@theme-ui/components'

[@pauliescanlon/gatsby-theme-gatstats](https://github.com/PaulieScanlon/gatsby-theme-gatstats) supports embedded images but even though the image is set to `display: inline-flex` by default `<p>` tags will be set to `display: block` meaning content won't 'float' next to the images. 

Since `.mdx` can render React components to achieve layouts like the one below it's time to implement [@theme-ui-components](https://theme-ui.com/components/) and use `Flex` and `Box` as you would in any `.js` file. 

In one `Box` you can add an `EmbeddedImage` and set its width to be 100% and allow the `Box` to do the hard stuff. 

Putting raw body text inside the other `Box` gets you half the way there but the font styling will be lost so wrap the text in a `<p>` to resolve this. 

The add some padding to the `Box` and boom, you're done!


The content of this page looks like this



```js
<Flex
  sx={{flexWrap: 'wrap'}}
>

  <Box
    sx={{
      p: 8,
      width:['100%', '50%' ]
    }}
  >
    <EmbeddedImage
      src={props.embedded.image1}
      width='100%'
    />
  </Box> 

  <Box
    sx={{
      p: 8,
      width:['100%', '50%' ]
    }}
  >
    <p>Pretium nibh id, porttitor nisl. Integer vitae metus arcu. Etiam ante nisl, ultrices non rutrum eu, porttitor eget diam. Ut venenatis purus ac sapien posuere fermentum. Praesent at lectus venenatis, lacinia tortor in, dignissim mi. Vestibulum mollis purus eu vehicula convallis. Donec ipsum arcu, accumsan vel quam quis, pretium volutpat sapien</p>
  </Box>

  <Box
    sx={{
      p: 8,
      width:['100%', '50%', '75%' ]
    }}
  >
  <p>Pretium nibh id, porttitor nisl. Integer vitae metus arcu. Etiam ante nisl, ultrices non rutrum eu, porttitor eget diam. Ut venenatis purus ac sapien posuere fermentum. Praesent at lectus venenatis, lacinia tortor in, dignissim mi. Vestibulum mollis purus eu vehicula convallis. Donec ipsum arcu, accumsan vel quam quis, pretium volutpat sapien</p>
  </Box> 

  <Box
    sx={{
      p: 8,
      width:['100%', '50%', '25%' ]
    }}
  >
    <EmbeddedImage
      src={props.embedded.image2}
      width="100%"
      justifyContent="center"
    />
  </Box>  

</Flex>
```
